# AI Media Platform Frontend

React + Vite frontend connected to Java Spring Boot microservices through API Gateway.

## Backend Gateway
Default backend URL:
```text
http://localhost:8080
```

## Run
```bash
npm install
npm run dev
```

Open:
```text
http://localhost:5173
```

## Pages
- Login
- Register as Creator or Consumer
- Feed
- Creator Upload Dashboard
- Search
- Profile
- Creator Analytics
